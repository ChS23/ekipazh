# Оформление в PDF

## Когда включается

- Пользователь просит "сделай PDF", "оформи красиво", "для клиента"
- После генерации КП — предложи: "Хочешь PDF для отправки клиенту?"
- Пользователь хочет напечатать или отправить документ

## Что делать

### 1. Определи артефакт

Если `$ARGUMENTS` не указан — найди последний созданный артефакт в `контент/`.
Если указан — прочитай его.

### 2. Определи формат

| Тип артефакта | Typst-формат | Стиль |
|---|---|---|
| КП (`контент/кп/`) | Документ | Деловой, чистый, 2 тарифа как таблица |
| Пост (`контент/посты/`) | Не нужен PDF | Скажи: "Пост — это текст для соцсетей, PDF не нужен. Скопируй текст." |
| Возражение (`контент/возражения/`) | Не нужен PDF | Скажи то же |
| Профиль (`профиль/`) | Документ | Сводка бизнеса, для себя или партнёра |
| Анализ (`анализ/`) | Документ | Отчёт со SWOT, таблицами, персонами |

### 3. Создай Typst-файл

Создай `.typ` файл рядом с артефактом. Используй чистый, профессиональный стиль:

**Основные принципы дизайна:**
- Шрифты: PT Sans (основной), PT Mono (код/цифры). Fallback на системные если нет
- Русский язык: `#set text(lang: "ru")`
- Щедрые отступы, воздух между блоками
- Без generic AI-эстетики — никаких градиентов, теней, скруглений везде
- Цветовая палитра: 1 акцентный цвет максимум, остальное — чёрный + серый + белый

**Структура КП:**
```typst
#set document(title: "Коммерческое предложение", author: "...")
#set page(paper: "a4", margin: (x: 2cm, y: 2.5cm))
#set text(font: "PT Sans", size: 11pt, lang: "ru")

// Шапка
// Понимание задачи
// Решение
// Тарифы (таблица)
// Почему мы
// Следующий шаг
// Контакты
```

**Структура отчёта (анализ/профиль):**
```typst
// Заголовок + дата
// Оглавление (если > 3 страниц)
// Секции из md-артефакта
// Таблицы для SWOT, конкурентов
// Персоны — каждая на своей странице
```

### 4. Компилируй

```bash
typst compile файл.typ файл.pdf
```

Если typst не установлен — скажи:
```
Typst не установлен. Установи: https://typst.app или `brew install typst`
А пока — вот .typ файл, можешь скомпилировать на typst.app онлайн.
```

### 5. Покажи результат

```
Готово: контент/кп/kp-марина.pdf

Хочешь поправить оформление?
```

## Важно

- Контент берётся из md-артефакта без изменений. Ты оформляешь, не переписываешь.
- Если в профиль/стиль.md есть визуальные предпочтения — учти (цвета, шрифты).
- Не добавляй лишнего: водяные знаки, "сгенерировано ИИ", лого Экипажа.
