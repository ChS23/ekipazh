# Состояние проекта

## Сканирование

Через Glob:
- `профиль/*.md`
- `анализ/*.md`, `анализ/персоны/*.md`
- `правила/*.md`
- `.claude/rules/*.md`
- `.claude/skills/**/SKILL.md` — пользовательские скиллы
- `контент/посты/**/*.md`, `контент/кп/**/*.md`, `контент/возражения/**/*.md`
- `решения/*.md`

Если `профиль/бизнес.md` есть — прочитай первые 20 строк для определения названия.

Если есть auto-memory (`~/.claude/projects/<current>/memory/MEMORY.md`) — прочитай её, чтобы ссылаться на накопленный контекст.

```

━━━ Профиль ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [статус] Бизнес
     [файлы или "не заполнен"]

  [статус] Анализ рынка
     [N персон, SWOT да/нет]

  [статус] Правила: [количество]
     [коротко: про что]

     [имена: /tg-post, /wa-reply]

  [статус] Контент:
     • Посты: [N черновиков, N опубликовано]
     • КП: [N черновиков, N отправлено, N принято]
     • Возражения: [N]

━━━ Решения ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [N] решений: [номера и названия]

━━━ Память ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [если есть memory — краткое "что помню про тебя"]

━━━ Что можно дальше ━━━━━━━━━━━━━━━━━━━━━

```

## Статусы

- `●` заполнено
- `◐` начато
- `○` не начато

## Проактивные подсказки в конце

На основе текущего состояния:

- Нет профиля → "Начни с разговора про бизнес — настрою за 15 минут."
- Профиль есть, нет правил → "Попробуй /post — посмотришь как звучит. После 2-3 корректировок оформим правило."
- Правил 3+, нет пользовательских скиллов → "Работаешь с Telegram/Instagram/WhatsApp? /workflow — сделаю скилл под твой канал."
- Много постов без структуры → "У тебя 10+ постов в одной папке. Разложить по идеи/черновики/опубликовано?"
- Есть КП в черновиках неделю → "КП для [клиент] в черновиках 7 дней. Отправил уже?"
- Давно не было активности → "Последнее действие N дней назад. Может быть нужна помощь с тем, что висит?"

## Если проект пустой

```
Экипаж v0.3.0 — Проект не настроен

Расскажи про свой бизнес — настроим систему за 15 минут.
Я задам вопросы, ты ответишь как есть — оформлю профиль, стиль, состояние.

Просто начни: "я делаю ..." или "я консультант по ..."
```
