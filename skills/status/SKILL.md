# Состояние проекта

## Что делать

1. Просканируй рабочую директорию через Glob:
   - `профиль/*.md` — ищет бизнес, стиль, состояние
   - `анализ/*.md` — рынок, конкуренты, swot
   - `анализ/персоны/*.md` — персоны и анти-персона
   - `правила/*.md` — правила
   - `контент/посты/*.md`, `контент/кп/*.md`, `контент/возражения/*.md`
   - `решения/*.md`

2. Если `профиль/бизнес.md` есть — прочитай первые 20 строк для определения названия бизнеса

3. Сформируй отчёт по формату ниже

## Формат вывода

```
Экипаж v0.2.0 — [название бизнеса или "Проект не настроен"]

━━━ Профиль ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [статус] Ваш бизнес
     [файлы или "не заполнен"]

  [статус] Анализ рынка
     [файлы или "не начат"]

  [статус] Правила
     [количество или "нет правил"]

  [статус] Контент
     [количество или "нет контента"]

━━━ Решения ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [количество] решений
  [список с номерами и названиями]

━━━ Скиллы ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  /start [бизнес]       — выгрузить бизнес в систему
  /analyze              — анализ рынка и аудитории
  /rule [задача]        — создать правило
  /post [тема]          — создать пост
  /kp [клиент]          — создать КП
  /objection [тема]     — обработать возражение
  /status               — этот отчёт
```

## Статусы

- `●` — заполнено (все обязательные артефакты)
- `◐` — начато, не завершено
- `○` — не начато

## Определение завершённости

- **Профиль заполнен:** есть все 3 файла (бизнес, стиль, состояние)
- **Профиль начат:** есть хотя бы 1 из 3
- **Анализ заполнен:** есть рынок + хотя бы 1 персона + конкуренты + swot
- **Анализ начат:** есть хотя бы 1 файл
- **Правила есть:** есть хотя бы 1 правило
- **Контент есть:** есть хотя бы 1 пост, КП или возражение

## Если проект пустой

```
Экипаж v0.2.0 — Проект не настроен

Расскажи про свой бизнес — настроим систему под тебя.
```
