# Редактирование артефакта

## Когда включается

- "Переделай вчерашний пост короче"
- "КП для Марины — сделай на 'вы'"
- "Адаптируй этот пост для Telegram"
- "Перепиши это возражение мягче"
- "Убери эмодзи из всех постов за неделю" (batch)

## Что делать

### 1. Определи цель

Какой артефакт редактировать:
- **Последний сгенерированный** (в текущей сессии) — если юзер говорит "этот", "последний", "вот этот"
- **По имени клиента/теме** — "КП для Марины", "пост про скидку"
- **По дате** — "вчерашний пост", "последнее КП"
- **Batch** — "все посты за неделю"

Glob `контент/**/*.md`, найди подходящие. Если несколько кандидатов — уточни.

### 2. Определи что изменить

Разбери запрос пользователя:

| Запрос | Изменение |
|--------|-----------|
| "короче", "сократи" | Убрать воду, оставить суть |
| "длиннее", "развёрни" | Добавить детали, примеры |
| "серьёзнее" / "смешнее" / "мягче" | Сменить эмоциональный тон |
| "убери эмодзи" / "добавь CTA" | Локальная правка |
| "на 'вы'" / "на 'ты'" | Сменить обращение |
| "для Instagram/Telegram/ВК" | Адаптировать формат под платформу |
| "применить правило NNN" | Взять `правила/правило-NNN.md` и переписать по нему |
| "как [стиль]" | Переписать в другом стиле из истории |

### 3. Загрузи контекст

1. Прочитай целевой артефакт
2. Прочитай `профиль/стиль.md`
3. Прочитай применимые правила
4. Если адаптация под платформу — учти ограничения (Telegram: 1000 символов, Instagram: визуал-ориентация)

### 4. Переписать

Сделай правку. Сохрани суть, меняй форму.

### 5. Покажи результат

Покажи **что было → что стало**:

```
Было:
[старый текст]

Стало:
[новый текст]

Изменено: [что именно]
```

### 6. Сохранить или вариант?

```
• "сохрани" → перезапишу файл, обновлю updated-дату
• "другой вариант" → ещё попытка
• "не то" → оставлю старый, расскажи что поправить
```

## Batch-правки

Если юзер говорит "убери эмодзи из всех постов за неделю":

1. Glob последние N артефактов
2. Покажи список: "Нашёл 5 постов. Применить ко всем?"
3. Жди подтверждения
4. Правь по одному, показывай прогресс
5. В конце: "Готово. Изменены: [список]"

## Метаданные

При сохранении добавь во фронтматтер:

```yaml
updated: YYYY-MM-DD
edit-reason: [что изменили]
```

Не перезаписывай `created`.

## Важно

- Не меняй суть. Если пользователь хочет другую суть — это новый артефакт через `/post` или `/kp`, не `/edit`
- Если правка затрагивает 5+ артефактов — предложи оформить как правило: "Хочешь чтобы в будущем так делалось автоматически? /rule"
- Сохраняй трассировку — добавь к списку источников "предыдущая версия артефакта"
