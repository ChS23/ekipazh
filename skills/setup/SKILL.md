# Настройка рабочего окружения

## Когда включается

- Первая настройка после `/start`
- Пользователь спрашивает "как удобнее каждый день открывать"
- Просит ярлык, иконку, быстрый запуск
- Жалуется что долго/неудобно запускать

## Что делать

### 1. Определи ОС

```
!`uname -s 2>/dev/null || ver`
```

- `Darwin` → macOS
- `Linux` → Linux
- `Microsoft Windows` или вывод `ver` → Windows

### 2. Скопируй подходящий скрипт

Плагин лежит в `${CLAUDE_PLUGIN_ROOT}`. Скопируй скрипт в текущую рабочую директорию пользователя:

| ОС | Скрипт | Куда |
|----|--------|------|
| macOS | `${CLAUDE_PLUGIN_ROOT}/scripts/start.command` | `./start.command` |
| Linux | `${CLAUDE_PLUGIN_ROOT}/scripts/start.sh` | `./start.sh` |
| Windows | `${CLAUDE_PLUGIN_ROOT}/scripts/start.ps1` | `./start.ps1` |

На Unix (macOS/Linux): `chmod +x start.*` после копирования.

### 3. Объясни как сделать ярлык

**macOS:**
```
Готово. Теперь чтобы запускать одним кликом:
1. Открой Finder в папке бизнеса
2. Правый клик на start.command → Создать псевдоним (alias)
3. Перетащи псевдоним на рабочий стол
4. Переименуй в "Экипаж"

Двойной клик — открывается терминал с Экипажем.
```

**Windows:**
```
Готово. Теперь чтобы запускать одним кликом:
1. Правый клик на рабочем столе → Создать → Ярлык
2. В поле "Расположение объекта":
   powershell.exe -NoExit -ExecutionPolicy Bypass -File "C:\полный\путь\start.ps1"
3. Назвать "Экипаж"
4. (Опционально) Свойства ярлыка → Сменить значок

Двойной клик — открывается PowerShell с Экипажем.
```

**Linux:**
```
Готово. Теперь:
1. chmod +x start.sh (уже сделано)
2. В файловом менеджере: Свойства → Разрешения → Разрешить запуск как программы
3. Двойной клик — запустится в терминале

Либо создай .desktop файл для меню приложений.
```

### 4. Рекомендации по IDE

Если юзер ещё не установил IDE:

```
Для удобной работы с файлами (профиль, правила, контент):

• Cursor (рекомендую) — cursor.com
  Встроенный AI, красивый UI, можно отключить если мешает
  
• VS Code — code.visualstudio.com  
  Бесплатный, расширение Markdown All in One для навигации

В IDE:
1. Открой папку своего бизнеса
2. Ctrl+` — встроенный терминал
3. В терминале: ./start.sh (или start.command)
```

### 5. Подтверждение

```
Настройка готова ✓

• start.[ОС] — скрипт запуска в папке бизнеса
• Ярлык на рабочем столе — клик → Экипаж открывается
• IDE (опционально) — визуальный просмотр файлов

Теперь каждый день: клик на "Экипаж" → говоришь что нужно.
```

## Важно

- Не перезаписывай существующий start.* если он уже есть — спроси юзера
- Если скрипт не скопировался (нет прав, другая причина) — дай инструкцию вручную
- На Windows без админ-прав может требоваться `Unblock-File` для .ps1 — упомяни если видишь ошибку
